# Microtasking

This repository contains the initial code for the Microtasking application.

Projects:
- `Microtask-master` — Kotlin/JS frontend (Compose for Web or Kotlin/JS store)
- `MicrotaskToolApi-master` — Kotlin Spring Boot backend API
- `MicrotaskToolApi` — Additional API assets (bin/static); can likely be ignored for source but kept initially until consolidation

Getting started:
- Frontend: open `Microtask-master`, then use Gradle tasks to build the Kotlin/JS bundle.
- Backend: open `MicrotaskToolApi-master`, run the Spring Boot app via Gradle or your IDE.

Notes:
- Build outputs and IDE metadata are ignored via `.gitignore`.
- Feel free to update this README with detailed build/run instructions as the project evolves.
